# 🐦 CyberCrow AI – Autonomous Cybersecurity Intelligence Podcast

CyberCrow AI is an autonomous cybersecurity intelligence system that:
- Collects real-time cyber threat news
- Generates AI-written podcast scripts
- Converts intelligence into human-like audio
- Sends live Telegram threat alerts
- Runs automatically on a schedule

Created by **Steven Bush**

---

## 🚀 Features

- 🧠 AI-powered cybersecurity news analysis
- 🎙 Automated podcast generation
- 🔔 Telegram real-time threat alerts
- 🔄 Fully automated daily execution
- 🐧 Built for Linux / Kali environments

---

## 🏗 Architecture

News Sources → AI Script Engine → Voice Generator → Podcast Output → Telegram Alerts
